Type 별 와인의 알코올 도수 평균

Type 별로 Alcohol 의 평균을 확인합니다.

wine %>% 
  group_by(Type) %>% 
  summarise(avg_alcohol = mean(Alcohol))

## # A tibble: 3 x 2
##   Type  avg_alcohol
##   <fct>       <dbl>
## 1 1            13.7
## 2 2            12.3
## 3 3            13.2

Type 별 와인의 알코올 도수 분포

Type 별로 Alcohol 의 분포를 Boxplot 으로 확인합니다.

library(ggplot2)

wine %>% 
  ggplot(aes(Type, Alcohol)) +
  geom_boxplot() +
  labs(title = 'Alcohol Boxplot by Type')

와인의 도수는 1번 타입이 높은 편이고, 2번 타입이 낮은 편입니다. 하지만 2번 타입 중 몇몇의 와인은 1번의 특정 와인보다 알코올 도수가 높은 와인도 존재함을 알 수 있습니다.
